■ インストールについて
  インストールディレクトリ名には空白文字、日本語などの文字列は含まれてはいけません。

  以下の環境変数の設定が必須です。
  MWSDK_ROOT :         SDK のルートディレクトリ (/ 区切り, 例: C:/Work/MWSDK/)
  MWSDK_ROOT_WINNAME : Windows のみ。
                       SDK のルートディレクトリ (\ 区切り, 例: C:\Work\MWSDK\)

  上記の環境変数設定用のバッチファイルを用意しています。
    SET_ENV.cmd      : 環境変数を設定する。
    UNSET_ENV.cmd    : 設定した変数を抹消する。


■ コマンドライン
  コマンドラインを起動するためのバッチファイルを用意しています。
  MWSDK 内の make コマンドを実行するための設定が含まれています。

  BASH.cmd           : bash (mingw の実行環境)
  CMD.cmd            : cmd.exe


■ ライセンスについて
  特に断りがない限り、当社製のソフトウェアについては MWSLA-1 (モノワイヤレスソフ
  トウェア使用許諾契約書) が適用されます。

  半導体メーカー製のライブラリ・gccなどのツールチェイン・bash (mingw) といった、
  当社製ではないソフトウェアについては、これらに付随するライセンスに従います。
  各個別のソースコードのコメントや、配布元のライセンス記述を参照してください。
